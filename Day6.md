# 三用電表
+ 大綱
    + 定義說明
        + 可量測三種數值：電壓、電流、電阻
        + 分為數位式、類比式(指針) 
        + 電位差：電流動的壓力
        + 電壓：伏特(V)
        + 電流：安培(A)
        + 電阻：歐姆
    + 外觀與功能介紹
        + 探棒
        + L型香插頭
        + COM接地(黑)
    + 注意事項
        + 要量測的值，要調整至大於量測值，否則三用電表會故障。

# 工具介紹&使用
+ 剝纜刀
+ 斜口鉗
+ 纜線剪(Cable cutter)
+ 打線器

# 編紮實作
1. 剝除外皮
2. 剪下遮蔽層、抗溼層
3. 簇群帶依顏色綁上打結
4. 依主色碼進行編紮

+ Out出線-導出信號
